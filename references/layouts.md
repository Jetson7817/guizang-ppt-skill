# Layout Templates · Style A

## 1. Hero — Opening Page

- Full-viewport slide
- Background: one of the Style A background images (full-bleed)
- Title: H1, centered or left-aligned, `Ink` color with warm paper shadow
- Subtitle: Body, centered below title
- Optional: small label (Mono, 12px) at top-left corner

## 2. Chapter Cover — Section Divider

- Full-viewport or 80% height
- Large chapter number: H1, 0.15 opacity, positioned as watermark
- Chapter title: H2, centered
- Background: subtle gradient or background image at 30% opacity

## 3. Data Big Number — Key Metrics

- 2×2 grid on desktop, stacked on mobile
- Each cell: large number (H1, 48–72px) + label (Caption, Mono)
- Number in `Ink`, label at 0.5 opacity
- Optional: subtle border between cells (hairline)

## 4. Image Grid — Visual Gallery

- Masonry or uniform grid
- Images with 2px border-radius
- Optional caption below each image (Caption, 0.5 opacity)
- Gap: 12px mobile / 24px desktop

## 5. Timeline — Process/History

- Horizontal on desktop, vertical on mobile
- Each point: dot (4px, `Ink`) + date (Mono) + description (Body)
- Connecting line: 1px hairline
- Alternating top/bottom placement on desktop

## 6. Closing — Final Page

- Full-viewport
- Centered: thank you / contact / QR code
- Background: same as hero (consistent bookend)
- Optional: social handles at bottom (Mono, 12px)

## Responsive Behavior

All layouts:
- Mobile-first (stack vertically)
- Breakpoint at 768px for tablet
- Breakpoint at 1024px for desktop
- Touch: swipe horizontal for navigation
- Keyboard: arrow keys for navigation
